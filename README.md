# tinyos-example-apps
Build and run tinyos example apps

# installing tinyos toolchain
```
yay -S nesc-git tinyos-tools-git
yay -S gcc49
```

# setting up
```
chmod +x ./script/setup.sh
./script/setup.sh
```
